1. Installare Node.js
   - Vai su: https://nodejs.org
   - Clicca sul pulsante verde "LTS".
   - Installa lasciando tutte le opzioni predefinite.

2. Scaricare il progetto
   - Clicca sul pulsante verde "Code" su GitHub.
   - Scegli "Download ZIP".
   - Estrai in una cartella, per esempio:
     C:\BotDiscord

3. Aprire il prompt dei comandi
   - Premi WIN + R.
   - Scrivi: cmd
   - Premi Invio.
   - Vai nella cartella:
     cd C:\BotDiscord

4. Installare i pacchetti
   npm install

5. Configurare il file .env
   - Copia il file: .env.example
   - Rinominalo in: .env
   - Aprilo con Blocco note e scrivi:
     TOKEN=tuo_token
     ALLOWED_USERS=tuo_id
     DEBUG=false

6. Avviare il bot
   npm start
